This is a simple Student Management System developed using Core Java and JDBC.
The project is built without using any frameworks.
It is completely developed using:
  1. Core Java
  2. JDBC (Java Database Connectivity)
  3. MySQL Database
The main purpose of this project is to understand how Java connects with a database and performs CRUD operations.

Project Structure :
The project follows a clean layered architecture:
  1. Controller Layer
  2. Service Layer
  3. DAO Layer
  4. Entity / DTO
  5. Database Configuration
This structure helps in separating business logic and database logic.


Features : 
  1. Add new student
  2. View student details
  3. Update student information
  4. Delete student records
  5. Database connection using JDBC
  6. Layered structure (Controller, Service, DAO, etc.)


How to Run it : 
1. Clone the repository
2. Configure MySQL database
3. Update database credentials in config file
4. Run the MainApplication file
